# YANG-Chat-Android
**YANG Chat安卓版本正式发布**

安卓版本由JAVA语言构造。

**需要自行更新密钥**


**YANG Chat-PC程序已发布，欢迎下载测试**

https://github.com/JasonYANG170/YANG-Chat

# 适配机型
支持Android7.0-Android13.0设备。

支持鸿蒙2.0-3.1设备。

适配深色模式。

适配平板，折叠屏设备
# 使用说明
1.输入测试密钥

2.输入问题，点击发送。
