## 🚧本项目已停止维护
## 功能
- ✅ ChatGPT3.1
- ✅ 间断对话
- ✅ 连续对话
- ✅ 云服务请求


# 适配机型
支持Android7.0-Android13.0设备。

支持鸿蒙2.0-3.1设备。

适配深色模式。

适配平板，折叠屏设备
# 使用说明
1.输入测试密钥

2.输入问题，点击发送。
